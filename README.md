# Bản dịch tiếng Việt: *Adversarial Examples in Modern Machine Learning: A Review*

Repository lưu trữ bản dịch tiếng Việt của bài khảo sát *Adversarial Examples in Modern Machine Learning: A Review* của Rey Reza Wiyatno, Anqi Xu, Ousmane Dia và Archy de Berker.

## Nội dung

Bài khảo sát trình bày tổng quan về mẫu đối kháng trong học máy, tập trung vào miền thị giác máy tính. Các chủ đề chính gồm phương pháp tạo mẫu đối kháng, tấn công trong thế giới thực, cơ chế phòng thủ và tính chuyển giao của mẫu đối kháng.
